# Sales Performance Dashboard (Power BI Project)

## Project Overview

This project presents an interactive Sales Performance Analysis Dashboard developed using Microsoft Power BI. The dashboard provides insights into sales trends across different regions, customer segments, product categories, and time periods. It helps users understand business performance through visual analytics and dynamic filtering.

---

## Objectives

The main objectives of this project are:

- Analyze overall sales performance
- Identify region-wise sales distribution
- Evaluate category and sub-category performance
- Detect top-performing products
- Analyze customer segment contributions
- Study monthly and yearly sales trends
- Provide interactive filtering using slicers

---

## Dataset Description

The dataset includes retail sales transaction details such as:

- Order ID
- Order Date
- Customer ID
- Customer Name
- Segment
- Region
- State
- Category
- Sub-Category
- Product Name
- Ship Mode
- Sales

The dataset represents sales activities across multiple regions in the United States.

---

## Tools and Technologies Used

- Microsoft Power BI Desktop
- CSV Dataset (Superstore Sales Data)
- Power Query for data cleaning
- DAX for calculated measures
- Interactive visualizations

---

## Dashboard Features

The dashboard consists of four analytical pages:

### 1. Overview Dashboard
- Total Sales KPI
- Total Orders KPI
- Total Customers KPI
- Average Sales KPI
- Sales by Region
- Sales by Segment
- Sales by Category

### 2. Sales Trend Analysis
- Monthly Sales Trend
- Year-wise Sales Performance
- Sales by Shipping Mode

### 3. Product Insights Dashboard
- Sales by Category
- Sales by Sub-Category
- Top 10 Products by Sales

### 4. Customer Analysis Dashboard
- Sales by Customer Segment
- Region-wise Segment Sales Distribution
- State-wise Sales Distribution Map
- Interactive slicers (Region, Category, Year)

---

## Key Insights

- West region generated the highest overall sales
- Consumer segment contributed the largest revenue share
- Technology category recorded the highest sales
- Standard shipping mode was most frequently used
- Top-performing products contributed significantly to total revenue
